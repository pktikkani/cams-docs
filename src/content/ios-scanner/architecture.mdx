## Architecture Pattern

**MVVM** with SwiftUI:
- ViewModels as `ObservableObject`
- `@Published` properties for reactive UI
- `async/await` for network operations
- Combine for event streams (camera delegate)

## Camera Pipeline

```
AVCaptureSession
    ↓
AVCaptureDeviceInput (back camera)
    ↓
AVCaptureVideoPreviewLayer → SwiftUI UIViewRepresentable
    ↓
AVCaptureMetadataOutput (delegate)
    ↓
metadataOutput(_:didOutput:) → QR string value
    ↓
ScannerViewModel.handleQRDetected(value:)
    ↓
API: POST /api/v1/scanner/validate
    ↓
Update @Published scanState → UI recomposition
```

## State Machine

```swift
enum ScanState {
    case scanning        // Camera active
    case validating      // API call in progress
    case result(Result)  // Granted or Denied
    case error(String)   // Network/parse error
}
```
